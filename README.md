# Информация о проекте:
**Название:** Y_Lab  
**Описание:** Проект от школы Y-Lab, в котором надо реализовать приложение на FastAPI с использованием PostgreSQL в качестве БД. В проекте следует реализовать REST API по работе с меню ресторана, все CRUD операции.
Даны 3 сущности: Меню, Подменю, Блюдо. 


# Запуск проекта:  
- Клонировать проект  
- Создать файл .env, где прописать данные из env.template  
- Установите необходимые пакеты, запустив "pip install -r requirements.txt`  
- Запустите приложение, выполнив команду в консоле **y_lab % uvicorn app.main:app --reloa**
